Key Technical Notes

  • –Wolf runs as root (uid 0) — XDG_RUNTIME_DIR (/tmp/sockets) owned by uid 1000
  • –Wolf socket: /tmp/sockets/wolf.sock — symlinked to /var/run/wolf/wolf.sock via systemd service
  • –AMP uses fixed MAC to prevent licence deactivation on restart
  • –Streaming client M.2 slot is CNVi only — standard PCIe cards won't work
  • –Your Client OS uses TDE, not KDE Plasma
  • –Docker Compose v2 (docker compose with space)
  • –Your Linux OS on GameStack Host runs Wayland by default
  • –GameStack is fully portable — tar the folder, move to any Linux + Docker + AMD GPU machine, run setup.sh
  • –Wolf app state folders use numeric session IDs as folder names
  • –Network is triple NAT: Internet → ISP Modem → Asus Router (YOUR_MODEM_IP) → Your Router (YOUR_UPSTREAM_ROUTER_IP / YOUR_ROUTER_IP) → GameStack Host
  • –Your Router is DMZ'd on Asus router — Asus forwards all unsolicited inbound to YOUR_UPSTREAM_ROUTER_IP
  • –Spectrum public IP is dynamic (currently YOUR_PUBLIC_IP) — DDNS setup required for reliability
  • –Any proxied service: keep proxied via Cloudflare — use .well-known delegation when federation needed
